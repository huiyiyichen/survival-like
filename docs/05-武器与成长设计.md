# 05 武器与成长设计

## 成长结构
成长系统由“升级三选一、4 武器 + 4 被动槽位、统一宝箱、双分支进化”四部分组成。武器和被动的最高等级都固定为 `5`，不再采用旧版的 8 级武器结构。

成长优先级明确分层：武器负责明显拐点与战斗手感，被动负责线性平滑地放大方向感。这样角色定位、单局节奏和宝箱规则可以彼此咬合，不会各写各的。

## 升级池
升级池默认给出三选一。前期轻度偏向初始武器，中后期再偏向已持有武器、已持有被动和它们的联动件。

| 规则 | 说明 |
| --- | --- |
| 前期偏向 | 前几级更容易出现初始武器和新组件，让开局更快成型 |
| 中后期偏向 | 持有武器后，更容易刷到该武器升级和对应联动被动 |
| 被动开放 | 六个被动从开局全部可出，不做角色专属被动池 |
| 满级排除 | 已满级武器和已满级被动不再进入升级池 |
| 补位规则 | 当有效强化不足 3 项时，只用魔晶补位，不再出现额外货币补位卡 |

升级补位魔晶分三档：前期 `10`、中期 `15`、后期 `20`。补位只在有效强化不足 3 项时出现，不会主动挤占正常成长选项。

## 宝箱规则
宝箱不分品质，不分外观，规则与 [03-单局流程设计](./03-单局流程设计.md) 完全一致。武器成长文档只负责解释开箱时如何结算成长。

1. 先检查所有满足条件的可进化武器。
2. 如果存在可进化项，本箱只负责进化，不再给额外等级。
3. 一个宝箱可以连续进化多件武器。
4. 无歧义的进化自动完成，双分支冲突时才弹出分支选择。
5. 如果没有可进化项，则按 `10% / 20% / 40% / 20% / 10%` 的概率给出 `1-5` 级随机强化总量。
6. 随机强化只命中当前持有且未满级的武器或被动，允许重复命中同一项。

## 武器骨架
本稿先冻结六把武器的 1 级起手、中段拐点和 5 级高潮。最终细数值以后续数值稿为准，但功能走向在这里视为冻结。

| 武器 | 1级起手 | 中段拐点 | 5级高潮 | 主要定位 |
| --- | --- | --- | --- | --- |
| 火球术 | 单发火球 + 小范围爆炸 | `2级` 与 `4级` 各增加一枚火球 | 爆炸半径显著扩大 | 爆裂范围群伤 |
| 回旋短刃 | 角色周围两枚短刃环绕 | `3级` 旋转速度明显加快 | 额外增加一圈刀刃 | 近身护体 |
| 冰羽法杖 | 小扇形三枚冰羽 | `3级` 额外增加一枚冰羽 | 扇形覆盖角度明显变宽 | 广域减速 |
| 雷鸣符文 | 定时落下一道链式闪电 | `3级` 弹射次数增加 | 弹射链数显著增加 | 连锁清群 |
| 荆棘种子 | 周围生成少量延迟爆开的毒荆区 | `3级` 毒区数量增加 | 再次提高毒区同时存在上限 | 持续毒伤 |
| 圣光锤 | 缓慢挥出一次高伤圣锤 | `3级` 单次命中伤害明显提高 | 单次命中伤害显著提升 | 重击爆发 |

## 进化分支
多路径进化保留，但所有进化条件统一为“武器满级 + 持有对应被动 + 开启宝箱”，不要求对应被动也满级。

| 武器 | 分支一 | 条件 | 分支二 | 条件 |
| --- | --- | --- | --- | --- |
| 火球术 | 烈焰法环 | 火球术 5 级 + 奥术棱镜 + 宝箱 | 陨火风暴 | 火球术 5 级 + 力量护符 + 宝箱 |
| 回旋短刃 | 飓风刃轮 | 回旋短刃 5 级 + 奥术棱镜 + 宝箱 | 影袭连刃 | 回旋短刃 5 级 + 疾风羽饰 + 宝箱 |
| 冰羽法杖 | 寒霜棱雨 | 冰羽法杖 5 级 + 奥术棱镜 + 宝箱 | 极寒风暴 | 冰羽法杖 5 级 + 冰封徽记 + 宝箱 |
| 雷鸣符文 | 链式雷暴 | 雷鸣符文 5 级 + 疾风羽饰 + 宝箱 | 天罚雷柱 | 雷鸣符文 5 级 + 力量护符 + 宝箱 |
| 荆棘种子 | 荆棘牢笼 | 荆棘种子 5 级 + 冰封徽记 + 宝箱 | 剧毒花海 | 荆棘种子 5 级 + 影袭披风 + 宝箱 |
| 圣光锤 | 守护裁决 | 圣光锤 5 级 + 生命圣杯 + 宝箱 | 震地圣锤 | 圣光锤 5 级 + 力量护符 + 宝箱 |

## 被动骨架
被动整体走线性平滑路线，不做像武器那样的满级高潮。只有离散属性允许保留少量里程碑，其余等级都按稳定增幅处理。

| 被动 | 成长重点 | 节奏说明 |
| --- | --- | --- |
| 力量护符 | 伤害 + 击退 | 伤害每级都加，击退隔级提升 |
| 疾风羽饰 | 攻速 + 移速 | 每级都同时加攻速和移速 |
| 奥术棱镜 | 投射数量 + 范围 | 范围每级都加，数量只在 `3级` 增加一次 |
| 冰封徽记 | 减速 + 控制时长 | 每级都同时增强减速和时长 |
| 生命圣杯 | 生命上限 + 回复 | 每级都同时增加生命和回复 |
| 影袭披风 | 暴击 + 持续伤害 | 暴击每级都加，持续伤害隔级提升 |

## 数值写法
成长表采用混合写法：伤害写具体值，冷却写秒数，持续效果写秒数，范围与加成写倍率或百分比。这样既能直观看懂，也不会让表格变成难读的纯参数墙。

## 扩展规则
新增武器时，先定义 1 级起手、3 级拐点和 5 级高潮，再补进化分支。新增被动时，先定义它服务哪两类武器，再决定是否需要离散里程碑。

1. 新武器必须能落回现有 12 分钟时间轴。
2. 新被动默认从开局可出，除非上游文档明确变更。
3. 任何成长规则变更都要先同步到 `03` 的成型节奏。
4. 任何进化规则变更都要同时回查 `03` 与 `07` 的开箱说明。
