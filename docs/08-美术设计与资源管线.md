# 08 美术设计与资源管线

## 美术方向
本项目整体采用 **像素风**，画面基调为“奇幻魔物入侵下的诅咒林地”。美术首要目标不是高细节，而是让角色、敌人、特效、统一宝箱和 Boss 预警在高密度战斗中保持清楚。

本轮先冻结风格、资源分类和 `nanobanana` 生成管线，不冻结最终 prompt 细化与批量出图排期。具体资源打磨放到后续玩法文档稳定后再展开。

## 视觉支柱
| 维度 | 设计要求 | 目的 |
| --- | --- | --- |
| 视角 | 俯视偏 3/4 视角 | 兼顾角色识别与地面技能表现 |
| 比例 | 角色头身比偏夸张，敌人轮廓明确 | 提高怪群中的识别度 |
| 颜色 | 中高饱和角色，中低饱和场景，高亮技能特效 | 保证战斗信息层次 |
| 轮廓 | 深色描边，重点单位保留外轮廓 | 防止怪群混读 |
| 光影 | 单侧主光源，不做复杂体积光 | 保持像素风干净 |

## 资源分类
| 类别 | 内容 | 首版范围 | 优先级 |
| --- | --- | --- | --- |
| 角色 | 战斗像素体、受击帧、死亡帧、角色立绘 | 3 名角色 | 高 |
| 敌人 | 普通敌人、精英、腐化术士、诅咒黑龙 | 4 普通、1 精英、1 中段Boss、1 终Boss | 高 |
| 武器特效 | 投射物、范围圈、命中特效、进化特效 | 6 武器及其分支 | 高 |
| 场景 | 地表、废墟、祭坛、枯木、路口 | 1 张主地图 | 中 |
| 交互物 | 经验晶体、统一宝箱、稀有回复结晶 | 3 类核心交互物 | 中 |
| UI | 面板框、按钮、武器图标、被动图标、Boss预警条 | 全流程界面 | 高 |

## 规格约束
| 资产 | 推荐基准 | 说明 |
| --- | --- | --- |
| 角色战斗体 | `32x32` 或 `48x48` | 新手角色可略大，保证清晰 |
| 普通敌人 | `24x24` 到 `40x40` | 依据职责区分体量 |
| 精英与中段Boss | `48x48` 到 `64x64` | 至少高于普通敌人一个层级 |
| 终Boss黑龙 | `96x96` 到 `128x128` | 需要强辨识轮廓 |
| 图标 | `16x16` 或 `24x24` | 武器和被动图标统一栅格 |
| 地块 | `16x16` 或 `32x32` | 地图纹理必须可重复拼接 |

## 色彩基调
这套像素风走“暗场景 + 亮技能 + 鲜明敌我对比”的路线，不走灰暗脏旧路线。这样更适合幸存者式高密度战斗，也更适合课程展示截图。

| 对象 | 主色方向 | 材质感 |
| --- | --- | --- |
| 学徒法师 | 红橙 + 金色点缀 | 布袍、法杖、水晶 |
| 流浪剑士 | 深蓝 + 银灰 | 皮甲、金属、旧布料 |
| 森林猎手 | 绿蓝 + 冰白 | 轻甲、羽饰、冰晶 |
| 普通敌人 | 紫灰、棕绿、骨白 | 肉体、骨骼、石质 |
| 腐化术士与黑龙 | 黑紫、猩红、深金 | 腐化纹路、符文、厚重甲壳 |
| 技能特效 | 火焰橙、寒冰蓝、雷电黄、圣光白 | 用高亮像素块强化冲击感 |

## 生成管线
美术资源默认依赖 [$multi-provider-media-generation](C:/Users/26361/.codex/skills/multi-provider-media-generation/SKILL.md) 的 `media-skill` CLI，并统一使用 `nanobanana`。由于 `nanobanana` 属于最小输入 provider，生成请求只传允许字段，不传 `size`、`steps`、`negative_prompt` 等禁用字段。

1. 先在对应设计文档里冻结需求，再生成图像。
2. 先确认 `nanobanana` 配置可用，再决定使用 `txt2img` 还是 `img2img`。
3. 用 `txt2img` 生成多张概念初稿，只筛方向，不直接追求成稿。
4. 选定方向后，用 `img2img` 回修轮廓、颜色和视角。
5. 人工检查是否满足像素风、可读性和项目设定，不合格直接淘汰。
6. 最终通过的资源再按类别入库，并记录来源文档编号。

## Prompt占位
当前只保留基础 prompt 结构，不在本轮冻结细化词库。后续如果需要大规模产图，再按角色、敌人、UI 三类分别扩写。

| 资源类型 | 基础模板 |
| --- | --- |
| 角色战斗体 | `pixel art, top-down 3/4 view fantasy survivor character, clear silhouette, readable combat sprite, [角色名], [服装特征], [武器]` |
| 敌人 | `pixel art monster enemy for survivor-like game, top-down 3/4 view, strong silhouette, [敌人名], [攻击特征], cursed forest theme` |
| Boss | `pixel art boss for survivor-like game, top-down 3/4 view, large silhouette, [Boss名], [核心机制视觉点]` |
| UI 图标 | `pixel art game icon, clean silhouette, centered object, [武器或被动名], fantasy style` |
